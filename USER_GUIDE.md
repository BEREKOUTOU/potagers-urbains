# Guide Utilisateur - Green City Grow Hub

## Bienvenue sur Green City Grow Hub !

Green City Grow Hub est une plateforme communautaire de jardinage urbain qui vous permet de découvrir, rejoindre et gérer des jardins partagés dans votre ville. Ce guide vous explique comment utiliser toutes les fonctionnalités de l'application.

## Premiers Pas

### Inscription

1. Cliquez sur "S'inscrire" dans le menu principal
2. Remplissez le formulaire avec :
   - Nom d'utilisateur unique
   - Email valide
   - Mot de passe sécurisé (8 caractères minimum)
   - Prénom et nom
   - Localisation (ville, région)
   - Biographie (optionnel)
3. Acceptez les conditions d'utilisation
4. Cliquez sur "Créer un compte"

### Connexion

1. Cliquez sur "Se connecter"
2. Entrez votre email et mot de passe
3. Cliquez sur "Se connecter"

## Gestion du Profil

### Modifier vos Informations

1. Allez dans "Profil" depuis le menu
2. Cliquez sur "Modifier le profil"
3. Vous pouvez modifier :
   - Prénom et nom
   - Biographie
   - Localisation et région
   - Photo de profil (fonctionnalité à venir)
   - Numéro de téléphone (fonctionnalité à venir)

### Préférences Utilisateur

Dans la section "Préférences" :
- Notifications par email
- Notifications push
- Résumé hebdomadaire
- Langue de l'interface
- Fuseau horaire

## Découverte des Jardins

### Parcourir les Jardins

1. Cliquez sur "Découvrir les jardins" dans le menu principal
2. Utilisez les filtres pour affiner votre recherche :
   - Par localisation
   - Par type de jardin
   - Par taille
   - Par disponibilité

### Voir les Détails d'un Jardin

Pour chaque jardin, vous pouvez voir :
- Description complète
- Localisation sur carte
- Membres actuels
- Événements à venir
- Photos du jardin
- Statistiques de croissance

### Rejoindre un Jardin

1. Cliquez sur un jardin qui vous intéresse
2. Cliquez sur "Rejoindre ce jardin"
3. Attendez la validation de l'administrateur du jardin
4. Une fois accepté, vous recevrez une notification

## Gestion des Jardins

### Créer un Nouveau Jardin

1. Allez dans "Mes jardins"
2. Cliquez sur "Créer un jardin"
3. Remplissez les informations :
   - Nom du jardin
   - Description
   - Localisation précise
   - Type de jardin (balcon, toit, communautaire, etc.)
   - Taille (en m²)
   - Nombre maximum de membres
   - Règles du jardin

### Gérer Votre Jardin

En tant qu'administrateur d'un jardin :

#### Membres
- Voir la liste des membres
- Accepter/refuser les demandes d'adhésion
- Modifier les rôles des membres
- Retirer des membres

#### Événements
- Créer des événements
- Gérer les inscriptions
- Modifier les événements existants

#### Statistiques
- Voir les statistiques du jardin
- Ajouter de nouvelles mesures
- Analyser les tendances de croissance

### Modifier un Jardin

1. Allez dans "Mes jardins"
2. Sélectionnez le jardin à modifier
3. Cliquez sur "Modifier"
4. Apportez vos changements
5. Sauvegardez

## Événements

### Parcourir les Événements

1. Cliquez sur "Événements" dans le menu
2. Filtrez par :
   - Date (à venir, passés)
   - Jardin
   - Type d'événement

### Créer un Événement

1. Allez dans "Mes jardins"
2. Sélectionnez un jardin dont vous êtes membre
3. Cliquez sur "Créer un événement"
4. Remplissez :
   - Titre
   - Description
   - Date et heure de début/fin
   - Lieu
   - Nombre maximum de participants
   - Type d'événement

### S'inscrire à un Événement

1. Cliquez sur un événement
2. Cliquez sur "S'inscrire"
3. Choisissez votre statut : "Je participe" ou "Peut-être"

### Gérer les Participants

En tant qu'organisateur :
- Voir la liste des inscrits
- Contacter les participants
- Modifier l'événement
- Annuler l'événement

## Discussions Communautaires

### Parcourir les Discussions

1. Cliquez sur "Communauté" dans le menu
2. Parcourez les discussions par catégories :
   - Conseils de jardinage
   - Échange de graines
   - Événements locaux
   - Questions générales

### Créer une Discussion

1. Cliquez sur "Nouvelle discussion"
2. Choisissez une catégorie
3. Donnez un titre accrocheur
4. Rédigez votre message
5. Ajoutez des tags si nécessaire
6. Publiez

### Participer aux Discussions

- Répondez aux messages
- Utilisez @ pour mentionner des utilisateurs
- Utilisez les réactions (like, utile, etc.)
- Signalez les messages inappropriés

## Ressources et Guides

### Accéder aux Ressources

1. Cliquez sur "Ressources" dans le menu
2. Parcourez par catégories :
   - Guides de plantation
   - Conseils d'entretien
   - Calendrier lunaire
   - Recettes avec les légumes du jardin

### Créer une Ressource

1. Cliquez sur "Ajouter une ressource"
2. Choisissez le type :
   - Article
   - Guide pratique
   - Vidéo
   - Lien externe
3. Remplissez les informations
4. Ajoutez des tags pour la recherche

### Guides Détaillés

Certains guides incluent :
- Étapes détaillées
- Photos d'illustration
- Conseils spécifiques à votre région
- Liens vers des ressources complémentaires

## Partage de Photos

### Uploader une Photo

1. Cliquez sur "Partager une photo"
2. Sélectionnez une photo depuis votre appareil
3. Ajoutez une description
4. Associez-la à un jardin (optionnel)
5. Choisissez la visibilité (publique ou privée)
6. Publiez

### Gérer vos Photos

- Modifiez la description
- Changez la visibilité
- Supprimez des photos
- Organisez en albums

## Statistiques

### Statistiques Personnelles

Dans votre tableau de bord :
- Nombre de jardins rejoints
- Événements participés
- Contributions à la communauté
- Évolution de votre activité

### Statistiques de Jardin

Pour chaque jardin :
- Croissance des plantes
- Rendements par saison
- Participation des membres
- Évolution de la biodiversité

### Ajouter des Mesures

1. Allez dans un jardin
2. Cliquez sur "Statistiques"
3. Cliquez sur "Ajouter une mesure"
4. Choisissez le type de mesure :
   - Taille des plantes
   - Rendement
   - Qualité du sol
   - Biodiversité

## Fonctionnalités IA

### Recommandations Personnalisées

L'IA vous aide à :
- Choisir les meilleures plantes pour votre jardin
- Optimiser l'arrosage selon la météo
- Prévoir les périodes de plantation
- Résoudre les problèmes de parasites

### Analyse de Photos

- Identifiez les plantes automatiquement
- Détectez les maladies des plantes
- Recevez des conseils de traitement

## Conseils d'Utilisation

### Bonnes Pratiques

1. **Communiquez régulièrement** avec les membres de votre jardin
2. **Partagez vos connaissances** dans les discussions
3. **Respectez les règles** de chaque jardin
4. **Contribuez aux événements** organisés
5. **Maintenez vos informations à jour**

### Sécurité

- Utilisez un mot de passe fort
- Ne partagez pas vos informations de connexion
- Signalez tout comportement inapproprié
- Respectez la vie privée des autres membres

### Support

Si vous rencontrez un problème :
1. Consultez ce guide utilisateur
2. Vérifiez les discussions communautaires
3. Contactez l'administrateur du jardin concerné
4. Ouvrez un ticket de support si nécessaire

## Raccourcis Clavier

- `Ctrl + /` : Recherche globale
- `Ctrl + N` : Nouvelle discussion
- `Ctrl + E` : Créer un événement
- `Ctrl + P` : Profil utilisateur

## FAQ

### Questions Fréquentes

**Q: Comment quitter un jardin ?**
R: Allez dans "Mes jardins", sélectionnez le jardin, cliquez sur "Quitter ce jardin".

**Q: Puis-je créer plusieurs jardins ?**
R: Oui, vous pouvez créer et gérer plusieurs jardins en tant qu'administrateur.

**Q: Les données sont-elles sécurisées ?**
R: Oui, toutes les données sont chiffrées et stockées de manière sécurisée.

**Q: Puis-je exporter mes données ?**
R: Oui, vous pouvez demander l'export de vos données personnelles.

**Q: Comment contacter l'équipe de support ?**
R: Utilisez le formulaire de contact dans la section "Aide".
